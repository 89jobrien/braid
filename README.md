# Braid

Rust-first workspace for the Braid personal agent platform.

Initial crates:
- `braid-model`: shared domain model
- `braid-core`: runtime engine and tool execution contracts
- `braid-providers`: provider adapter boundary with a mock implementation
- `braid-cli`: operator entrypoint for local runs
